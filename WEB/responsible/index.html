宇宙最強スペック

私が初めて手にしたデスクトップPCは、 DELL から販売されている ALIENWARE シリーズである ALIENWARE X51 という、コンパクトながらもPCゲームにおいて抜群のパフォーマンスを叩き出す、宇宙最強スペックのものでした。４年前は。

気になるそのスペックですが、CPU：Intel(R)Core(TM)i7-3770 メモリ:8G HDD:1T グラフィックボード:GeForce GTX555 となっており、購入してから４年間どのゲームにおいてもPCスペックで困ったことは無いです。

しかしながら欠点も多少あり、一つの売りであるコンパクトさという点を実現させるためか、冷却部分に少し不安があり、夏場での長時間の起動は熱暴走の危険があるため注意しなければなりません。しかしながら日頃から埃を取り除くなどの手入れをしっかりとしていれば、かなりの長期間私たちゲーマーの期待に答えてくれるPCとして働いてくれます。

マウスはもうこれ以外使えません

ゲーマーたるもの、PCの周辺機器にもかなりこだわりを持つべきだと私は考えます。そんな私の右腕とも言える相棒であるマウスは、知る人ぞ知る Razer の DeathAdder シリーズです。

こちらのシリーズはかなり愛用者も多く、 DeathAdder201X といったように年度ごとに新商品が出るほどのマウスになっています。私が初めて手にしたのは DeathAdder2011 で、３ヶ月でホイールをダメにしながらも、あまりの使いやすさに１年間使い続けました。

その後、買い換えるときに選んだのが DeathAdder2011 でした。そう、同じものを買い求めました。そして２代目も使い潰し、今では最新作である DeathAdder CHROMA を使っています。ここまで同じシリーズのものを使い続けているせいか、なかなか他のマウスに挑戦することができなくなってしまいました。

高ければ高いほどいい

キーボードの話をさせてください。「ゲームするのにキーボードは別に安くてもよくない？そんなに変わらないでしょ」と言う人、多いと思います。そんなことはないです。私が今現在使っているのは12000円で買った、FILCO の Majestouth2 Tenkeyless 青軸 です。高ければ高いほどいいです。

実は私自身も、ゲームパッド（コントローラーのようなもの）を主に使用していた時期があり、キーボードなんて別に安くても、と思っていた時期がありました。しかしながら遊ぶゲームのジャンルが、いわゆるMMORPGに傾きつつある中で考えがガラリと変わりました。そのゲームのものにもよりますが、私が長く遊んだゲームの中には、左手だけで常に数十個のキーを正確に順序良く押してプレイするものがありました。そこで安いキーボードを使用していた頃は、指への負担、ストレスが大きかったのを今でも覚えています。

ですがキーボードを買い替えた途端、スイッチを押しているような感覚、キーを押す速さ、ゆえに現れるゲームへの対応スピードが劇的に変わりました。せっかくいいものを手に入れたということで、日頃のメンテナンスも欠かさなくなりました。いいこと尽くしです。

音を制する者はゲームを制す

PCゲームをプレイするとき、通話もするためにヘッドセットをつける方が多いです。私はこれ、 steelseries の siberia v2 Full-size Headset です。ゲームをプレイする前提でヘッドセットを選ぶ時に大切なのは、自分の頭に合う形かどうかです。この siberia は、他のものと比べるとフィットしやすく多くの方に好まれて使われているものの一つになります。頭の形大きさは人それぞれなので、疲れにくい合うものを探しましょう。

滑る派、止まる派

マウスパッドには、大きさだけでなく、滑りやすさ、止まりやすさがあるのをご存知でしょうか。いわゆるFPSをプレイする人にとってはマウス、もしくはそれ以上にこだわりを持つ人が多いです。そのせいか本当に種類も多く、一番個性の出てくるところだと思います。今現在私が使っているのは ARTISAN の 疾風シリーズ の 疾風 乙 XSOFT です。止めやすさにかなり重点を置いており、止めたいところでピタリと止まるものになっています。ちなみにこちらのマウスパッド、我らがプロゲーマーほそごえくんから頂きました。ありがとうございます。
